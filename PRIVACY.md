# DiyyMotion Privacy

DiyyMotion processes selected video files locally inside the browser.

- Metadata Patch and HQ Encode do not upload video files to a DiyyMotion server.
- Settings and processing counters are stored in Chrome local extension storage.
- Creator statistics and original download links are read from the TikTok page currently opened by the user.
- Repost removal sends requests directly to TikTok using the account already signed in within the browser.
- The TTWID fix removes only the TikTok `ttwid` cookie when the user presses the tool button.
- DiyyMotion does not include account authentication, subscriptions, analytics tracking, or an external creator database.
